Task Boards

Previously, there was just one task board containing all the tasks that you created. Starting with this release, you can create multiple task boards. This lets you organize tasks according to projects or certain time periods (e.g., product releases). You can also tie Central task boards to Jira boards (see Jira Integration).

If you already have tasks on your license, you will notice that they have all been placed in a default board. If you do not already have tasks on your license, there is no default board; instead, you need to first create a board and then add tasks to it.

How to Delete a Task Board

Deleting a task board cannot be undone. All tasks in the board will also be deleted. However, any messages and references to the task board will be retained in the Tasks Activity.

If you delete a task board in Central, the associated board on Jira remains.

The various page views associated with tasks (Board, Grid, Calendar, Backlog, Archive) are unique for each task board.

Also, a left arrow at the top lets you navigate back to the main grid listing all task boards.
